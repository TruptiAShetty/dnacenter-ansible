.. Title

cisco.dnac.nfv_provision -- Resource module for Nfv Provision
+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. Collection note

.. note::
    This plugin is part of the `cisco.dnac collection <https://galaxy.ansible.com/cisco/dnac>`_ (version 3.1.6).

    To install it use: :code:`ansible-galaxy collection install cisco.dnac`.

    To use it in a playbook, specify: :code:`cisco.dnac.nfv_provision`.

.. version_added

.. versionadded:: 1.0.0 of cisco.dnac

   :local:
   :depth: 1

Synopsis
--------

.. Description

- Manage operation create of the resource Nfv Provision.

.. note::
    This module has a corresponding :ref:`action plugin <action_plugins>`.
