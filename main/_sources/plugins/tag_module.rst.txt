.. Title

cisco.dnac.tag -- Manage Tag objects of Tag
+++++++++++++++++++++++++++++++++++++++++++

.. Collection note

.. note::
    This plugin is part of the `cisco.dnac collection <https://galaxy.ansible.com/cisco/dnac>`_ (version 2.0.7).

    To install it use: :code:`ansible-galaxy collection install cisco.dnac`.

    To use it in a playbook, specify: :code:`cisco.dnac.tag`.

.. version_added

.. versionadded:: 1.0.0 of cisco.dnac

.. contents::
   :local:
   :depth: 1

.. Deprecated


Synopsis
--------

.. Description

- Returns the Tags for given filter criteria.
- Creates Tag with specified Tag attributes.
- Updates a Tag specified by id.
- Deletes a Tag specified by id.
- Returns Tag specified by Id.
- Returns Tag count.

.. note::
    This module has a corresponding :ref:`action plugin <action_plugins>`.

.. Examples

Examples
--------

.. code-block:: yaml+jinja

    
    - name: get_tag
      cisco.dnac.tag:
        state: query  # required
        additional_info_attributes: SomeValue  # string
        additional_info_name_space: SomeValue  # string
        field: SomeValue  # string
        level: SomeValue  # string
        limit: SomeValue  # string
        name: SomeValue  # string
        offset: SomeValue  # string
        order: SomeValue  # string
        size: SomeValue  # string
        sort_by: SomeValue  # string
        system_tag: SomeValue  # string
      register: nm_get_tag

    - name: create_tag
      cisco.dnac.tag:
        state: present  # required
        description: SomeValue  # string
        dynamicRules:
        - memberType: SomeValue  # string
          rules:
            values:
            - SomeValue  # string
            items: None
            operation: SomeValue  # string
            name: SomeValue  # string
            value: SomeValue  # string
        id: SomeValue  # string
        instanceTenantId: SomeValue  # string
        name: SomeValue  # string
        systemTag: True  # boolean

    - name: update_tag
      cisco.dnac.tag:
        state: present  # required
        description: SomeValue  # string
        dynamicRules:
        - memberType: SomeValue  # string
          rules:
            values:
            - SomeValue  # string
            items: None
            operation: SomeValue  # string
            name: SomeValue  # string
            value: SomeValue  # string
        id: SomeValue  # string
        instanceTenantId: SomeValue  # string
        name: SomeValue  # string
        systemTag: True  # boolean

    - name: delete_tag
      cisco.dnac.tag:
        state: absent  # required
        id: SomeValue  # string, required

    - name: get_tag_by_id
      cisco.dnac.tag:
        state: query  # required
        id: SomeValue  # string, required
      register: nm_get_tag_by_id

    - name: get_tag_count
      cisco.dnac.tag:
        state: query  # required
        count: True  # boolean, required
        attribute_name: SomeValue  # string
        level: SomeValue  # string
        name: SomeValue  # string
        name_space: SomeValue  # string
        size: SomeValue  # string
        system_tag: SomeValue  # string
      register: nm_get_tag_count
