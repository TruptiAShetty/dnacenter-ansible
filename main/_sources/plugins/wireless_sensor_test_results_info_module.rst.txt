.. Title

cisco.dnac.wireless_sensor_test_results_info -- Information module for Wireless Sensor Test Results
+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. Collection note

.. note::
    This plugin is part of the `cisco.dnac collection <https://galaxy.ansible.com/cisco/dnac>`_ (version 3.1.0).

    To install it use: :code:`ansible-galaxy collection install cisco.dnac`.

    To use it in a playbook, specify: :code:`cisco.dnac.wireless_sensor_test_results_info`.

.. Examples

Examples
--------

.. code-block:: yaml+jinja

    
    - name: Get all Wireless Sensor Test Results
      cisco.dnac.wireless_sensor_test_results_info:
        dnac_host: "{{dnac_host}}"
        dnac_username: "{{dnac_username}}"
        dnac_password: "{{dnac_password}}"
        dnac_verify: "{{dnac_verify}}"
        dnac_port: "{{dnac_port}}"
        dnac_version: "{{dnac_version}}"
        dnac_debug: "{{dnac_debug}}"
        siteId: string
        startTime: 0
        endTime: 0
        testFailureBy: string
      register: result
