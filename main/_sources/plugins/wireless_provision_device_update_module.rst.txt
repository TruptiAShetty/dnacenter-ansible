.. Title

cisco.dnac.wireless_provision_device_update -- Resource module for Wireless Provision Device Update
+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. Collection note

.. note::
    This plugin is part of the `cisco.dnac collection <https://galaxy.ansible.com/cisco/dnac>`_ (version 3.1.7).

    To install it use: :code:`ansible-galaxy collection install cisco.dnac`.

    To use it in a playbook, specify: :code:`cisco.dnac.wireless_provision_device_update`.

Synopsis
--------

.. Description

- Manage operation update of the resource Wireless Provision Device Update.

.. note::
    This module has a corresponding :ref:`action plugin <action_plugins>`.
