.. Title

cisco.dnac.event_series_info -- Information module for Event Series
+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. Collection note

.. note::
    This plugin is part of the `cisco.dnac collection <https://galaxy.ansible.com/cisco/dnac>`_ (version 3.0.0).

    To install it use: :code:`ansible-galaxy collection install cisco.dnac`.

    To use it in a playbook, specify: :code:`cisco.dnac.event_series_info`.

.. version_added

.. versionadded:: 1.0.0 of cisco.dnac

   :local:
   :depth: 1

Synopsis
--------

.. Description

- Get all Event Series.

.. note::
    This module has a corresponding :ref:`action plugin <action_plugins>`.

Examples
--------

.. code-block:: yaml+jinja

    
    - name: Get all Event Series
      cisco.dnac.event_series_info:
        dnac_host: "{{dnac_host}}"
        dnac_username: "{{dnac_username}}"
        dnac_password: "{{dnac_password}}"
        dnac_verify: "{{dnac_verify}}"
        dnac_port: "{{dnac_port}}"
        dnac_version: "{{dnac_version}}"
        dnac_debug: "{{dnac_debug}}"
        eventIds: string
        startTime: 0
        endTime: 0
        category: string
        type: string
        severity: string
        domain: string
        subDomain: string
        source: string
        offset: 0
        limit: 0
        sortBy: string
        order: string
      register: result
